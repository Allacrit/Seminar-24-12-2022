Для создания коммита используется команда *git commit*. Для этого в терминале с папкой-репозиторием необходимо написать *git commit <сообщение к коммиту>*. Сообщение к коммиту писать ***ОБЯЗАТЕЛЬНО!! И в ковычках!!!***.
